# genpw

Password Generator

## Usage

```
Usage of genpw:
  -n int
    	number of characters (default 16)
  -nc int
    	minimum count of numbers (default: any) (default -1)
  -sc int
    	minimum count of symbols (default: any) (default -1)
```

## Installation

```
go get github.com/mattn/genpw
```
